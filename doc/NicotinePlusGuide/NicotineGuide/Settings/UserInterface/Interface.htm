2.3.1 Nicotine+ Settings: Interface


Interface settings allows you to configure some Interface option if you need to


Close buttons on tabs

If you want to have close buttons on tabs

Close-buttons on chat room tabs Close-buttons on private chat tabs
  • default is checked

Show Buttons in Transfer Tabs

if you want GTK icons on buttons in down/upload tabs
  • default is checked

Display Tray Icon

Not available for windows

allows you to show Tray Icon when Nicotine+ is running.
  • default is checked

Chat font

allows you to configure font-family, style and size used in Nicotine+ chat/private tabs
  • default is GTK default.

Decimal seperator

Icon and Sound Themes Directory

Icon Theme Directory

Select a directory with images and Nicotine+ will use it for online/offline/away/hilite indication.
  • default is empty

Sound notification

Sound notification, when someone mentions your username
in public chatroom or PM's you. 2 different sounds are used.
If you want use sound notification, you need to meet Nicotine+ Optional Dependencies

Enable Sound Effect

if you don't want to use sound notification, you can uncheck it.
  • default is checked

Sound Effect command

choose between play -q or ogg123 -q
  • default is play -q

Sound Effects Directory

allows you to choose your own sound for use of sound notification.

Chat colors

which color messages get displayed in Chat room or Private chat

Chat color settings are only for your comfort, other users won't see any of you configs.
  • Remote text is the messages you get to see, when something is wrong with files you're downloading
  • default is empty
    • It shows up like Remote: [message]
      • Remote: User limit of xx megabytes exceeded
      • Remote: File error
  • Local text are your messages, you see them in Chat room and Private chat sessions
  • default is BLUE
  • /me text is used to state what you're doing instead of talking
  • default is FORREST GREEN
    • "/me is going to eat something" shows up as "[time][user] is going to eat something"
    • 01:07:01 * [user] is testing
    • 21:06:55 * [user] checks timestamps and stops expecting an answer
  • Highlighted text is a message that includes your username
  • default is red
    • this occurs when somebody is talking to you only, in a busy chatroom

Search colors

To make searching and downloading files easier, you can choose to color your search results
  • Immediate download shows black
  • default is empty
  • With queue downloads grey
  • default is GREY
